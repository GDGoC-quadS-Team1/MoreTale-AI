# outputs viewer

```bash
python3 outputs/viewer/server.py --host 127.0.0.1 --port 8787
```

브라우저에서 `http://127.0.0.1:8787/viewer/` 로 접속하면 됩니다.
드롭다운에서 `outputs/*_story_*` 실행 폴더를 선택해 페이지별 본문/오디오/일러스트를 확인할 수 있습니다.

- 메인 뷰어 우상단에서 페이지 선택, 자동재생, PDF 화면 이동을 사용할 수 있습니다.
- 퀴즈가 생성된 실행 폴더는 `http://127.0.0.1:8787/viewer/quiz.html?run=<run_id>` 에서 문제, 선택지, 정답, 해설, 출처 페이지를 책별로 확인할 수 있습니다.
- 자동재생은 현재 페이지부터 `원문 오디오 -> 번역 오디오 -> 다음 페이지` 순서로 끝까지 진행합니다.
- `http://127.0.0.1:8787/viewer/print.html?run=<run_id>` 에서 표지+본문 전체를 책 형태로 미리보고 브라우저의 PDF 저장/인쇄를 사용할 수 있습니다.
- 인쇄 레이아웃은 표지 뒤부터 `A4 세로 1장에 동화 2페이지` 가 들어가도록 압축되어 있습니다.
